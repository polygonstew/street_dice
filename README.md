<h2>Rules:</h2>
<p>After watching the yt video sited below, this is what I have came up with so far.</p>
<ol>
 <li>Find Players</li>
 <li>Set Shooter</li>
 <p>pick HIGH or LOW</p>
 <li>Throw the Dice</li>
 <p>who ever rolls it first is the shooter</p>
 <li>!! - Bet</li>
 <p>Shooter throws down / opponent can match or fade.</p>
 <li>MATCH and FADE</li>
 <p>MATCH is to match the bet. FADE is to</p>
 <li>Shooter ROLLS</li>
 <p>if it is 7 or 11 shooter wins / if it is 2, 3, or 12 loses the pot</p>
 <p>all other numbers will be set as POINT</p>
